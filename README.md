# Full Stack Portfolio Generator

A dynamic full-stack web application for generating and managing portfolio content with user authentication.

## Features
- User authentication (login/logout)
- Role-based access control
- Create, read, update, delete (CRUD) portfolio content
- Admin controls vs user-specific portfolios

## Tech Stack
PHP • MySQL • HTML/CSS/JS

## Setup
1. Clone the repo
2. Run `contact.sql` to set up database
3. Configure `db.php` with your database credentials
4. Start local PHP server (`php -S localhost:8000`)
